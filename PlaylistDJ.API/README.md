# Running backend in dev mode

Run these commands to start the backend service.

1. **Install dependencies**

```bash
yarn install
```

2. **Start MongoDB**

```bash
sudo mongod
```

3. **Run backend**

```bash
yarn dev
```

App is hot reloaded
