# Running frontend

Run these commands to start the backend service.

Install dependencies

```bash
yarn install
```

Start MongoDB

```bash
sudo mongod
```

## Development

```bash
yarn dev
```

App is hot reloaded

## Production

```bash
yarn build
yarn start
```
