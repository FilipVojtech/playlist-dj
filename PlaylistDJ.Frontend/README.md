# Running frontend in dev mode

Run these commands to start frontend.

```shell
yarn install
yarn dev
```

App is hot-reloaded using [Rollup](https://rollupjs.org) when editing scripts in the `src` directory.

# Building frontend

```shell
yarn install
yarn build
```
